Java Game

This java game is slightly different than a typical game of Pig. For starters, this game uses 2 dice. Rolling a one still ends the turn.

The user plays against a computer player. The computer player plays differently based on which strategy is selected.

Computer Strategies

Cautious Strategy

Using the Cautious Strategy, the computer will always choose to hold after one successful roll. "Slow and steady wins the race" is the cautious computer's motto.

Greedy Strategy

Using the Greedy Strategy, the computer will always attempt to roll 3 times, if possible. The greedy computer loves rolling the dice, metaphorically and digitally.

Random Strategy

Using the Random Strategy, the computer will sometimes roll and sometimes hold. You never know what you'll get with the random computer.

Options

Menu Selection Usage
File New Starts a new game with the same first player.
Exit Quits the game and closes the program.
Options Auto-Roll Sets the computer player to automatically roll the dice and take its turn.
Set Goal Score Sets the Goal Score for the game. The default score is 100.
Strategies Cautious Sets the Cautious Strategy
Greedy Sets the Greedy Strategy
Random Sets the Random Strategy
